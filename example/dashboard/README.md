# Dashboard Simulator

This is a DOMECS demo of an industrial hydraulic lift control dashboard.

## Features
- Lift height
- Visualize four hydraulic cylinders
  - Pressure gauge per cylinder
  - Extend/retract buttons per cylinder
- Emergency stop button
- Reset button
- Keyboard controls:
  - `QWER` to extend/retract cylinders
  - `Space` for E-stop
  - `R` for reset
  - `H` to input a target height (PID control to achieve the height by actuating the cylinders)
